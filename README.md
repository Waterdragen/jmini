# jmini
Layout analyzer bot [cmini](https://github.com/apsu/cmini) in java  
> cmini is a basic discord bot that displays the stats of a particular layout  

### How to run
> [!NOTE]
> - Java 15+ installed  
> - Maven installed
1. `mvn clean install`
2. `mvn exec:java`
